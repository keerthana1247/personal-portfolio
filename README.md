# My Portfolio

A personal portfolio website built with HTML, CSS, JS, Node.js + Express.

## Features
- Responsive design
- Contact form connected to MongoDB
- Runs locally on port 3000

## How to run
1. `npm install`
2. `npm start` 
3. Open http://localhost:3000 in your browser

## Tech Stack
HTML, CSS, JavaScript, Node.js, Express, MongoDB
